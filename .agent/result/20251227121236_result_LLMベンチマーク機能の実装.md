# LLMベンチマーク機能の実装

本タスクでは、Gemini ChessにおけるLLMベンチマーク機能を実装しました。
これにより、AIモデル間（Gemini 2.5 vs Gemini 3など）の対戦を自動化し、結果を収集してチェス能力を測定することが可能になりました。

## 主な変更点
1.  **ベンチマークモードの実装**:
    - 指定した回数の試合を自動で連続実行する機能。
    - 試合結果（勝者、決着理由、手数、PGN）の自動記録。
    - 結果のJSONダウンロード機能。
2.  **UIの追加**:
    - ゲーム画面下部にベンチマーク設定パネルを追加（試合数設定、開始/停止ボタン、進捗表示）。
